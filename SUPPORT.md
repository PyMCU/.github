# Support

PyMCU is community-driven and experimental.

- **Questions / Ideas**: open a GitHub issue (Discussion board may be enabled later).
- **Bugs**: use the “Bug report” template with repro steps.
- **Roadmap Feedback**: open a feature request and reference the roadmap section on the website.

If you wish to support development, you can donate via **Ko-fi** (see the “Sponsor” button on the org page).
