# PyMCU Project 🐍⚡

**PyMCU** is an **open-source, experimental AOT compiler** that translates a safe subset of Python into **ASM or C** for microcontrollers — aiming for deterministic, bare-metal execution.

- Deterministic by design: no OS, no GC.
- Experimental roadmap guided by the community.
- MIT-licensed.

## Roadmap (high level)
- **Late Q4 2025** — PIC10F/12F (ASM)
- **Early Q1 2026** — AVR8 / ATmega (C with ASM option)
- **Mid Q2 2026** — ARM Cortex-M (C with ASM option)
- **Late Q3 2026** — ESP32 (C with ASM option, Wi-Fi stubs)

> Future architectures will be prioritized based on community input.
